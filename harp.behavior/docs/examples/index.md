# Behavior Examples

Here you will find a collection of examples demonstrating how to use the `harp.behavior` Python package. Each example includes a brief description and the necessary code to get you started.

- [Analog Input](./behavior/analog_input/analog_input.md) - read values from a potentiometer.
- [Camera Trigger](./behavior/camera_trigger/camera_trigger.md) - trigger frames from a camera.
- [Digital Input](./behavior/digital_input/digital_input.md) - read values from a switch.
- [Digital Output](./behavior/digital_output/digital_output.md) - control an LED.
- [Digital Output Pulse](./behavior/digital_output_pulse/digital_output_pulse.md) - trigger a digital pulse.
- [LED Toggle](./behavior/led_toggle/led_toggle.md) - control the brightness of a LED by using the current driver ports.
- [PWM Toggle](./behavior/pwm_toggle/pwm_toggle.md) - control the brightness of a LED by using a PWM signal.
- [Quadrature Encoder](./behavior/quadrature_encoder/quadrature_encoder.md) - get the position values of a quadrature encoder.
- [Servo Motor Control](./behavior/servo_motor_control/servo_motor_control.md) - control a servo motor.
